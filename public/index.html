Crypto Icons API

This project builds ontop of the amazing Cryptocurrency Icons project.

Rather than bundling all the images into our app, Quids, and having to ship an update everytime a new currency was added we built this service.

API

GET https://cryptoicons.org/api/:style/:currency/:size

Styles

  • black
  • white
  • color
  • icon

Examples

GET https://cryptoicons.org/api/color/eth/600

GET https://cryptoicons.org/api/black/btc/200
